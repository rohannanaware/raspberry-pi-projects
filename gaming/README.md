# Retro Gaming Projects

This section contains documentation for various retro gaming operating systems and emulation platforms tested on different hardware.

## Projects

### [Batocera](./batocera/)
Linux-based retro gaming distribution for creating a dedicated gaming system. Boots entirely from USB/SD card.

### ReCalbox *(Coming Soon)*
Another Linux-based retro gaming distribution with a focus on ease of use.

### RetroPie *(Coming Soon)*
Raspberry Pi-focused retro gaming platform built on top of Raspbian.

## Overview

These projects explore different retro gaming solutions that can transform old hardware into dedicated gaming consoles. Each distribution has its own strengths:

- **Batocera**: Universal, boots from USB, works on x86 and ARM devices
- **ReCalbox**: User-friendly, similar to Batocera with different UI
- **RetroPie**: Raspberry Pi optimized, highly customizable

## Hardware Tested
- Dell Inspiron 15R (2011) - x86_64 laptop
- Raspberry Pi *(planned)*
